# API для YamDB

### Описание

Сервис YaMDb собирает отзывы пользователей на книги, фильмы или песни/другие музыкальные произведения. Сами произведения в YaMDb не хранятся, здесь нельзя посмотреть фильм или послушать музыку.

Пользователи могут оставлять текстовые отзывы о произведениях и ставить им оценку. На основе оценок пользователей формируется рейтинг каждой книги, фильма или песни.

Полная документация API: http://127.0.0.1:8000/redoc/

### Технологии

Проект написан с использованием Python 3.7 и Django REST Framework.

### Как запустить проект локально

Клонируйте репозиторий:

```git clone https://github.com/tanja-ovc/api_yamdb_group_project.git```

Убедитесь, что находитесь в директории _api_yamdb_group_project/_ либо перейдите в неё:

```cd api_yamdb_group_project/```

Cоздайте виртуальное окружение:

```python3 -m venv venv```

Активируйте виртуальное окружение:

* Для Linux/Mac:
 
    ```source venv/bin/activate```

* Для Windows:

    ```source venv/Scripts/activate```

При необходимости обновите pip:

```pip install --upgrade pip```

Установите зависимости из файла requirements.txt:

```pip install -r requirements.txt```

Примените миграции:

```cd api_yamdb/```

```python3 manage.py migrate```

Запустите проект:

```python3 manage.py runserver```

### Заполнение базы данных

Заполнение БД прилагающимися данными в формате .csv можно произвести с помощью команды ```python manage.py import_data```.

### Примеры запросов к API

Просмотр списка произведений (доступно всем):

```GET``` ```http://127.0.0.1:8000/api/v1/titles/```

Создание нового произведения (доступно админам, суперпользователям):

```POST``` ```http://127.0.0.1:8000/api/v1/titles/```

Просмотр отдельного произведения (доступно всем):

```GET``` ```http://127.0.0.1:8000/api/v1/titles/1/```

Изменение/удаление отдельного произведения (доступно админам, суперпользователям):

```PATCH/DELETE``` ```http://127.0.0.1:8000/api/v1/titles/1/```

Просмотр списка отзывов к отдельному произведению (доступно всем):

```GET``` ```http://127.0.0.1:8000/api/v1/titles/1/reviews/```

...

Изменение/удаление отзыва к отдельному произведению (доступно автору отзыва, модераторам, админам, суперпользователям):

```PATCH/DELETE``` ```http://127.0.0.1:8000/api/v1/titles/1/reviews/1/```

...

Просмотр списка комментариев к отдельному отзыву (доступно всем):

```GET``` ```http://127.0.0.1:8000/api/v1/titles/1/reviews/1/comments/```

...
